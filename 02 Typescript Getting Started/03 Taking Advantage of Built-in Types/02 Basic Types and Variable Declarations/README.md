#### Declarations with let and const

```javascript
console.log(someString);
var someString = 'Hello World';

// compiler error
console.log(someString);
let someString = 'Hello World';

// Good way
let someString = 'Hello World';
console.log(someString);

// constant value
let someString = 'Hello World';
console.log(someString);

```

- JS all variable declarations on top of the JavaScript called hoisting
